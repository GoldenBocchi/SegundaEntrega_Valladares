Segunda Entrega

Por Brian Valladares

Explicación del trabajo

El presente trabajo es un lector de arrays, el cual ejecutara la lectura de un array para luego ir seleccionando los distintos elementos y ejecutar operaciones como:

  • Generar arrays especificos mediante el metodo push
  • Realizar filtrado y concatenacion
  • Uso de funciones y muestra de mensaje

Todo esto mediante una consola interactiva, primero se indicara un mensaje mostrando todas la opciones.

Los pasos son los siguientes

  • Se muestra el mensaje de inicio indicando opciones
  • Se genera un prompt indicando que se indique la opcion
  • Luego de la selección y respuesa, se pregunta si se desea seguir trabajando
